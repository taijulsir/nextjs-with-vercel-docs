# Creating a Next.js Application

Next.js provides a streamlined process for creating new projects with a CLI tool called `create-next-app`.

## What is Next.js?
Next.js is a React framework that enables features like server-side rendering (SSR) and static site generation (SSG) out of the box. It’s perfect for everything from simple blogs to high-performance e-commerce sites.

## Initializing the Project

Open your favorite terminal and run the following command to start the creation process:

```bash
npx create-next-app@latest my-awesome-app
```

![Initial prompt for Next.js app](/images/create-next-app-prompt-1.jpeg)

### Choose Your Options
When prompted, we recommend the following configuration for a modern, production-ready application:

- **TypeScript?**: Yes (Recommended for type safety)
- **ESLint?**: Yes (Keeps your code clean)
- **Tailwind CSS?**: Yes (A utility-first CSS framework for rapid styling)
- **src/ directory?**: Yes (Maintains a clean root folder)
- **App Router?**: Yes (Recommended for the latest Next.js features)
- **import alias (@/*)?**: Yes (Easier file imports)

![Configuration selections](/images/create-next-app-prompt-2.jpeg)

To see your app in action, start the development server:

```bash
npm run dev
```
