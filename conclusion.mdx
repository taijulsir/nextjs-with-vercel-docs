## Troubleshooting Common Issues

If you run into any trouble during your deployment process, here are some common solutions:

### Build Errors
- Check your local terminal for linting or TypeScript errors (`npm run build`).
- Verify that your node version matches Vercel’s (default is Node 18+).

### Missing Environment Variables
- Ensure that the environment variable names in your code match those in the Vercel dashboard.
- Remember to push your changes to GitHub to trigger a fresh build after adding variables.
